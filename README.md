# Proyecto ABP

¡Bienvenido a mi repositorio! Este proyecto es parte de mi proceso de aprendizaje y desarrollo en el mundo de la programación.

## 🚀 Descripción
Este proyecto consiste en una aplicación de gestión de transacciones bancarias desarrollada con HTML, CSS y JavaScript, enfocada en una interfaz moderna y funcional.

## 🛠 Tecnologías utilizadas
- **HTML5** y **CSS3**
- **JavaScript**
- **Git/GitHub**

## 📂 Estructura del proyecto
- `index.html`: Estructura principal.
- `style.css`: Estilos (Glassmorphism).
- `script.js`: Lógica de transacciones.

## 💡 Aprendizajes clave
- Implementación de diseño responsivo.
- Manejo de ramas y resolución de conflictos en Git.
- Control de versiones.
